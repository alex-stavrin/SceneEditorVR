# SceneEditorVR

## About

SceneEditorVR is a simple to use 3D scene editor made for VR. 

## Video

[![SceneEditorVR](https://img.youtube.com/vi/ZL4JPvvvf-M/maxresdefault.jpg)](https://www.youtube.com/watch?v=ZL4JPvvvf-M)

## Features

- Add assets to scene from themed libraries
- Translate/Rotate/Scale assets
- Select and multi select assets
- Transform snapping
- Locomotion (Horizontal and Vertical)
- Snap turning
- Radial menus that allow you to:
  - Change transform mode
  - Delete assets
  - Duplicate assets
  - Undo
  - Redo
- Save your scene as readable .json file
- Export your file to .glb format

## Tech

It was developed using Unity 6000.3.10f. It runs easily on Meta Quest 2.
